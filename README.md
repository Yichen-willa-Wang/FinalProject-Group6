# FinalProject-Group6
Final project repository for GWU NLP course group work.
This repository contains all materials for the GWU NLP final group project.

## Repository Structure

- **Group-Proposal/**  
  Contains the final group proposal in PDF format.

- **Final-Group-Project-Report/**  
  Contains the final group project report in PDF format.

- **Final-Group-Presentation/**  
  Contains the final group presentation in PDF format.

- **Code/**  
  Contains all codes used in the final project, including data processing, modeling, and evaluation scripts.  
  This folder also includes its own `README.md` file explaining the purpose of each script and the recommended order for running the code.

## How This Repository Works

This repository is organized to match the course submission requirements.

1. The **Group-Proposal** folder stores the proposal PDF.
2. The **Final-Group-Project-Report** folder stores the final written report PDF.
3. The **Final-Group-Presentation** folder stores the presentation PDF.
4. The **Code** folder stores all project-related scripts and documentation.

All final deliverables are placed into their corresponding folders so that the instructor can easily review the project materials.

## Code Usage

Please refer to `Code/README.md` for details about:
- what each script does
- the order in which the scripts should be run
- any dependencies or notes for reproduction
